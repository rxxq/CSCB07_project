[2] click run (select AVD and project = FlyBook) 

/*********************HOW TO LOGIN *******************************************/
[A] use default admin account 
	  User : admin
	  Pass:  admin 
	  
	  *from here you can upload CSV files by going to admin options 
	    -> upload CSV file 
		-> upload client/flight info from CSV by specifying path

[B] register a new user (so you can surf as a client)
    + press register, fill out the fields and login with your email and pass
    
 
 /**************************HOW TO SEARCH FLIGHTS*****************************************/
AS CLIENT: 
	  + press search Flights button and fill out fields your prompted for
	  + press book to book any flights you see, and press shopping cart icon in actionbar
	    (in client menu) to view booked items

AS ADMIN: 
	  + go to regular optiosn menu, press search Flight Manifest
	  + fill out fields as prompted
	  
/*************************SEARCH ITINERARIES ***********************************************/
 AS CLIENT: 
	  + press search Avail. Itineraries button and fill out fields your prompted for
	  + press sortByCost or sortByTime button to sort itineraries by cost/ total travel time respectivly
	  + press book to book any itineraries you see, and press shopping cart icon in actionbar
	    (in client menu) to view booked items

AS ADMIN: 
	  + go to regular optiosn menu, press search Itineraries 
	  + fill out fields as prompted
	  (similar to client)

/**************************EDIT INFO (SELF) *********************************************/
AS CLIENT & ADMIN: 
	  + press the icon with a gear in actionbar, and edit your information that way
	  (changes may only  appear once app is restarted)
	  
	 
/************************VIEW BOOKED ITINERARIES/FLIGHTS ****************************************/
AS CLIENT & ADMIN:
	  + press the icon (shopping cart) to view booked itineraries & Flights
	  (AVAIL IN CLIENT & ADMIN MAIN MENU's
	  
/************************BOOK ITINERARIES/FLIGTS****************************************/
	  @see how to searching flights / searching itineraries 

/*************************UPLOAD CLIENT/FLIGHT INFORMATION (CSV FILES)*******************/
ADMIN ONLY:
	  + in the admin options spinner (in admin menu after admin logs in)
	    -> press upload CSV file
	    -> enter CSV file path and press flight info/ client info button to upload
	       respective types of csv files
	    -> if not presented with an error, file was saved succesfully
